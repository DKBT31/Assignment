🎮 Hướng Dẫn Tích Hợp Assets Tùy Chỉnh

Chúc mừng! Game engine đã được cập nhật để hỗ trợ assets tùy chỉnh của bạn. Làm theo hướng dẫn dưới đây để thay thế các hình ảnh mặc định.

📂 1. Background Image (Hình nền)

File cần tạo: assets/images/background.png
Kích thước khuyến nghị: 1400x800 pixels
Mô tả: Hình nền chiến trường Điện Biên Phủ
Cách sử dụng: Lưu hình nền pixel art của bạn (cảnh hoàng hôn/chiến trường) vào thư mục assets/images/ với tên background.png
Hình nền tùy chỉnh sẽ hiển thị ở đây

✈️ 2. Aircraft Sprite (Hình máy bay)

File cần tạo: assets/images/aircraft.png
Kích thước khuyến nghị: 80x32 pixels
Mô tả: Sprite máy bay địch (sẽ tự động lật theo hướng bay)
Cách sử dụng: Game sẽ tự động lật hình để máy bay bay đúng hướng. Thiết kế máy bay hướng về bên phải.
Sprite máy bay tùy chỉnh sẽ hiển thị ở đây

🔫 3. Gun Body (Thân pháo)

File cần tạo: assets/images/gun-body.png
Kích thước khuyến nghị: 60x40 pixels
Mô tả: Thân pháo phòng không (phần đế và khung)
Cách sử dụng: Thay thế phần thân pháo được vẽ bằng gradient. Nòng pháo sẽ được vẽ riêng và có thể xoay.
Thân pháo tùy chỉnh sẽ hiển thị ở đây

🎯 4. Gun Barrel (Nòng pháo)

File cần tạo: assets/images/gun-barrel.png
Kích thước khuyến nghị: 40x12 pixels
Mô tả: Nòng pháo (sẽ xoay theo mouse)
Cách sử dụng: Thiết kế nòng pháo nằm ngang. Game sẽ tự động xoay theo hướng ngắm.
Nòng pháo tùy chỉnh sẽ hiển thị ở đây

📋 Hướng Dẫn Chi Tiết

Bước 1: Tạo thư mục

Đảm bảo thư mục assets/images/ đã tồn tại trong game folder.

Bước 2: Thêm hình ảnh

Copy các file PNG của bạn vào thư mục với đúng tên:

  • background.png - Hình nền
  • aircraft.png - Máy bay
  • gun-body.png - Thân pháo
  • gun-barrel.png - Nòng pháo

Bước 3: Test game

Refresh trang game để xem assets mới. Nếu không load được, game sẽ dùng hình vẽ mặc định.

Lưu ý kỹ thuật:

  • Sử dụng định dạng PNG để có nền trong suốt
  • Kích thước có thể điều chỉnh nhưng nên giữ tỷ lệ
  • Hình ảnh sẽ được scale tự động
  • Game sẽ fallback về graphics cũ nếu không tìm thấy file

🎨 Các Tính Năng Đã Thêm

  • ✅ Hệ thống load ảnh bất đồng bộ
  • ✅ Fallback tự động nếu ảnh không load được
  • ✅ Tự động flip máy bay theo hướng bay
  • ✅ Giữ nguyên hiệu ứng muzzle flash và particles
  • ✅ Tương thích với hệ thống xoay nòng pháo
  • ✅ Console log để debug việc load ảnh
